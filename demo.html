Step-by-Step App Pairing

Follow these steps to connect the SmartWalker mobile application with a User and a Companion.

For User

Step 1

Go to Settings.

Step 2

Scan the QR code of your Companion to pair the devices.

Step 3

Alternatively, manually add a Companion using the Companion ID and Secret Code then Save.

For Companion

Step 1

Go to settings and go to Pairing section.

Step 2

Scan the QR code of the User to complete pairing.

Step 3

Alternatively, manually add the User using their User ID and Username then Save.
